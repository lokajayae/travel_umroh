# NTI Latihan Kelas
## Overview
Sebuah modul untuk Odoo 14 untuk manajemen kelas yang didalamnya terdapat Mahasiswa, Dosen, Mata Kuliah, dan Kelas. Struktur menu yang dibuat adalah sebagai berikut
 * Contacts
   * .....
   * Dosen
   * Mahasiswa
 * Kelas
   * Kelas
   * Mata Kuliah
